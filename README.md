# YouTube Force Max Quality

Forces preferred quality (including Premium enhanced bitrate) and theatre mode on every YouTube video.

## Install

[**Chrome Web Store**](https://chromewebstore.google.com/detail/youtube-force-max-quality/blcglfghdocllffdjgnbfhnenkeadeim)

## Features

- Set preferred video quality from 144p to 4K (2160p), or just pick "Max" for the highest available
- Automatically enable theatre mode
- Premium enhanced bitrate support (VP9, AV1)
